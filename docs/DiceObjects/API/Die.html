Die#

class cse587Autils.DiceObjects.Die.Die[source]#

A class used to represent a dice with n faces, each with probability p.

Parameters:

face_probs (list of float) – The probabilities of the faces

Examples:

>>> face_probs = [1/6]*6
>>> my_die = Die(face_probs)
>>> len(my_die)
6

>>> # access the probability of a face. Remember that python is zero-indexed!
>>> my_die[0]
0.16666666666666666

>>> # accessing a probability that is negative or outside of the range of the
>>> # die will raise an IndexError
>>> my_die[7]  
Traceback (most recent call last):
...
IndexError

>>> # rolling the die will return a random face
>>> # remember python is 0 indexed AND the upper bound is exclusive, meaning
>>> # in order to get the numbers 0, 1, 2, 3, 4, 5, you need to pass in the
>>> # range 0, 6
>>> my_die.roll() in set(range(0, 6))
True

likelihood(observed_data)[source]#
Calculate the likelihood of the observed data given the Die

face probabilities (face_probs).

Parameters:

observed_data (NDArray[numpy.int_]) – A list of observed face counts where the index of each element corresponds to the face, and the count is the number of times that face was observed. The sum of the counts is the number of times the die was rolled.

Returns:

The likelihood of the observed bin counts given the face probabilities.

Return type:

float

Raises:

  • TypeError – If the face counts is not a numpy array or a base python list.

  • ValueError – If the face counts is an empty list.

Examples:

>>> import numpy as np
>>> face_probs = [1/4]*4
>>> my_die = Die(face_probs)
>>> observed_data = np.array([1]*4)
>>> round(my_die.likelihood(observed_data), 4)
0.0039
