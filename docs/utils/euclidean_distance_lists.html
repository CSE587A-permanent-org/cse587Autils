euclidean_distance_lists#

cse587Autils.utils.euclidean_distance_lists.euclidean_distance_lists(list1, list2)[source]#

calculate the euclidean distance between two lists of numbers

Parameters:

  • list1 (list or numpy.ndarray) – first list of numbers

  • list2 (list or numpy.ndarray) – second list of numbers

Returns:

euclidean distance between the two lists

Return type:

float

Raises:

  • TypeError – if either list is not a list

  • ValueError – if the lists are not the same length

Example:

>>> euclidean_distance_lists([1, 2, 3], [1, 2, 3])
0.0

>>> euclidean_distance_lists([1, 2, 3], [4, 5, 6])
5.196152422706632
